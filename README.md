# BE Project

Spring Boot application with Java 21 and Spring Boot 3.5.4

## Prerequisites
- Java 21
- Maven 3.6+

## Running the application

```bash
mvn spring-boot:run
```

The application will start on port 8080.

## API Endpoints

- GET `/` - Returns "Hello World" JSON response

Example response:
```json
{
  "message": "Hello World"
}
```
